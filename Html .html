<DOCTYPE>

Defines the document type and version of HTML.

<!DOCTYPE html>

<html>

The root element of an HTML document.

<html lang="en"> ... </html>

<head>

Contains metadata, links to stylesheets, and the page title.

<head> ... </head>

<body>

Contains the main content of the webpage.

<body> ... </body>

<h1 to h6>

Defines headings from largest (h1) to smallest (h6).

<h1>Heading</h1>

<p>

Defines a paragraph of text.

<p>This is a paragraph.</p>

<a>

Creates a hyperlink.

<a href="https://example.com">Visit</a>

<img>

Embeds an image into the webpage.

<img src="image.jpg" alt="Description">

<ul><li>

Creates an unordered list.

<ul><li>Item 1</li></ul>

<form>

Creates a form for user input.

<form> ... </form>

<table>

Creates a table with rows and columns.

<table><tr><td>Data</td></tr></table>

<input>

Defines an input field for user data entry.

<input type="text" placeholder="Enter your name">

<button>

Defines a clickable button.

<button>Click Me</button>

<nav>

Defines a section of navigation links.

<nav><a href="#">Home</a> <a href="#">About</a></nav>

<footer>

Defines the footer section of a webpage.

<footer>Copyright Info</footer>

<span>

Used for styling or grouping inline content.

<span>Styled Text</span>

<section>

Defines a thematic grouping of content.

<section>Content Goes Here</section>
